Tutorial of Use

To use this software, you need open your browser console (Chrome is F12) and enter the commands in the command line.

You can make a new Deck using the proper command to this.

List of commands

  1. var deck = new Deck() - Makes a new Deck to the var deck
  2. deck.buildDeck() - Populates the Deck with Cards (Aces to K and their respective suits)
  3. deck.shuffleCards() - Shuffle the Deck Cards Randomly
  4. deck.getNonThrown() - Returns the number of non-throwed cards
  5. deck.getBottom() - Returns the bottom card value and suit
  6. deck.getTop() - Returns the top card value and suit
  7. deck.deal() - Delivers 3 cards and shows in console their respective value and suit
  8. deck.isThrown(0-51) - Check if the specified card was Throwed
  9. deck.getSuit(0-51) - Returns the specified card suit
  10. deck.getValue(0-51) - Returns the specified card value
  11. deck.printDeck() - Prints the whole deck in console
  12. deck.printCard(0-51) - Prints the specified card in console
